Thumbs Up Plugin adds social sharing buttons that are most popular in Poland.

== Description ==
Wtyczka umożliwia dodawanie przycisków NK Fajne!, Facebook Like oraz Google Plus+ w Twoim WordPress'ie.
Wyróżnienię treści za pomocą przycisku skutkuje pojawieniem się informacji o tych portalach.

Wtyczka nie ma jeszcze konfiguratora, ale bardzo łatwo zmienić parametry przycisków w kodzie.
Ręczna konfiguracja umożliwia określenie:

* Kolorystyki przycisków
* Układu poszczególnych przycisków
* Miejsc w których mają się pojawiać przyciski

== Installation ==

1. Umieść pliki wtyczki w katalogu `/wp-content/plugins/thumbs-up`
2. Włącz wtyczkę
3. Skonfiruguj przyciski - W każda klasa PHP (np. `thumbs-up-google.php`) zawiera zmienną $settings, gdzie znajdują się parametry przycisku.
4. Określ miejsce wyświetlania się przycisków - zmienna $settings w pliku `thumbs-up.php`.
5. Dostosuj style do własnych potrzeb.. Przyciski są opakowane w div'a o odpowidniech klasach - np. przycisk Facebook Like ma class="fb-like".
